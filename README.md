# ⚡ Revan's Learning Tracker

A personal progress tracker to monitor my coding journey across multiple tracks.

## 📚 Tracks Included

- 🎨 **HTML & CSS** — Web fundamentals (Completed ✅)
- ⚡ **DSA (C++)** — Data Structures & Algorithms
- 🌐 **JavaScript** — From basics to advanced concepts
- 🗄️ **DBMS** — MongoDB, SQL & Full Stack Libraries

## ✨ Features

- ✅ Topic-wise progress tracking with checkboxes
- 📊 Live progress bar and stats per track
- 📝 Upload & save notes for each section
- 👁️ Viewer mode — anyone can view progress
- 🔐 Owner mode — only Revan can mark topics & manage notes
- 🔒 Password-protected tracks (JS & DBMS)
- 📱 Mobile-friendly responsive design

## 🛠️ Built With

- HTML, CSS, Vanilla JavaScript
- LocalStorage for data persistence
- No frameworks, no backend — 100% frontend
